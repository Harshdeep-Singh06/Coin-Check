# 🚀 Crypto Check

A React-based cryptocurrency tracking app that shows real-time prices, charts, and market data using the CoinGecko API.

---

## 🌐 Live Demo

👉 coin-check-two.vercel.app

---


## ✨ Features

* 🔍 Search cryptocurrencies
* 📊 Sort by price, rank, and market cap
* 📈 Interactive 7-day price chart
* 💰 View market cap, volume, and supply
* 🔗 Dynamic routing for coin details

---

## 🛠️ Tech Stack

* React
* Tailwind CSS
* Recharts
* CoinGecko API

---

## 🚀 Run Locally

```bash
npm install
npm run dev
```
